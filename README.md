# 🏥 Jenan Surgical Speciality Center

> **Yaalii Baqaqsanii Yaaluu Ol'aanaa Jennaan — We Serve You!!**

A premium, ultra-modern medical website for **Jenan Surgical Speciality Center** — Ethiopia's premier surgical and specialty clinic located at Shaka Mall, Ashewa Meda, Burayu, Addis Ababa.

---

## ✨ Features

- **Homepage** — Animated hero with floating 3D medical cross, live stat counters, service cards, facilities gallery, Why Choose Us, team section, and CTA
- **About Us** — Story, mission & vision, core values, milestone timeline, and team profiles
- **Services** — Detailed showcase of all 6 core services with facility photos and feature lists
- **Contact** — Appointment booking form, embedded Google Maps, 24/7 emergency callout

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Gemeking/jenan-website.git
cd jenan-website

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
